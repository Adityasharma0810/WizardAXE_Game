# WizardAXE Game

A small Unity game project named "WizardAXE". This repository contains the Unity project files and a packaged build (RAR) of the game.

## Contents

- Assets/ — Unity project Assets folder (scenes, scripts, shaders, art, etc.)
- ProjectSettings/ — Unity project settings
- Packages/ — Unity package manifest
- How to play.txt — short instructions for running the packaged build
- WizardAxegame.rar — packaged build archive (extract and run the included application)

## Unity Version

This project includes ProjectSettings/ProjectVersion.txt which lists the editor version:

- m_EditorVersion: 6000.2.10f1

Open the project with the Unity Editor that matches this version if possible. If you don't have that exact version, try using the closest available Unity Editor version; Unity Hub can help install the version.

Note: If Unity warns about upgrading the project when you open it, create a backup before allowing automatic upgrades.

## Getting started (open in Unity)

1. Clone the repository:

   git clone https://github.com/Adityasharma0810/WizardAXE_Game.git

2. Open Unity Hub and choose "Add" (or "Open") and select the cloned project folder.
3. Let Unity import assets and compile scripts. This may take a few minutes on the first open.
4. Open the main scene (look inside Assets/Scenes or search for a scene asset) and press Play in the Editor to run the game.

## Run the packaged build

1. Download WizardAxegame.rar from the repository.
2. Extract the archive to a folder on your machine.
3. Run the executable inside the extracted folder (Windows build: .exe; macOS: open the .app bundle). See `How to play.txt` for the same short instruction.
